# About
This is a common location for stock photos used by Enlace One. It's also the folder for Adobe Stock contributions, though there is no cross-over between those two groups of Icons and Adobe Stock contributions are not actually synced to GitHub. 

# Adobe Stock

## Creation SOP

Use [an SVG editor like this one](https://freesvgeditor.com/en/svg-editor-online). 

(!) Size it before you start if exporting as a PNG. If you forget, just group it then expand

(!) Name it in the editor (Defaults to "drawing" on the right) (Only works for SVGs:/ )

(!) If uploading as an SVG, then move text several clicks to the left and check in Adobe before uploading (it gets moved right for some reason)

Download with a transparent background as 4MP-45MP, for example:
- 2000 x 2000
- 1000 x 4000
- 4096 × 1024
- 3500 x 1500

## Upload SOP

Check the box next to "Its an Icon" if it's simple enough to be used in a UI. (Only works for SVGs :/ )
